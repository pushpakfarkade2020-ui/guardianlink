# 🚨 GuardianLink – SOS Emergency Safety App

GuardianLink is a fast and reliable Android safety application designed to help users send instant emergency alerts with their live location.

---

## 📱 Features

- 🔴 One-tap SOS emergency button  
- 📍 Sends live location via Google Maps link  
- 📇 Add & manage emergency contacts  
- 🧩 Home screen widget for quick access  
- ⚡ Lightweight and fast performance  
- 📳 Vibration feedback on SOS trigger  

---

## ⚙️ How It Works

1. User taps the **SOS button**
2. App fetches current GPS location
3. Generates a Google Maps link
4. Opens SMS app with pre-filled message
5. User sends alert to emergency contacts

---

## 🛠️ Tech Stack

- **Language:** Java  
- **Platform:** Android Studio  
- **Location API:** FusedLocationProviderClient  
- **Storage:** SharedPreferences  
- **UI:** XML Layouts  

---


---

## 🔐 Permissions Used

- ACCESS_FINE_LOCATION – To fetch user location  
- SEND_SMS (optional) – For emergency alerts  
- FOREGROUND_SERVICE – For background operations  

---

## ⚠️ Disclaimer

GuardianLink is designed for emergency assistance purposes only.  
Users should ensure proper permissions are granted for the app to function correctly.
